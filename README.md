# mizpos2 - Serverless POS system for Dojin sales

同人誌とか売る時用のいい感じに完全クラウド運用できる POS システムです。

## ディレクトリ構成

基本的にバックエンドとインフラをまとめて管理するモノレポです。

```
.
├── lambda // バックエンドのLambdaコード
├── terraform
│   ├── modules // 基本的に本番と開発は同じ環境なのでそれに合わせたデプロイ設定
│   ├── tf-dev // 開発環境関連のデプロイ設定
│   └── tf-prod // 本番環境関連のデプロイ設定
└── zig // デプロイツールの設定とか
```

ここにない詳細は、各ディレクトリの中の README にあるかも。
